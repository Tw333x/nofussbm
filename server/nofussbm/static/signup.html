No Fuss Bookmarks

No Fuss Bookmarks is a very simple bookmarks store designed for hackers (that don't need fancy interfaces, but nice APIs).

The data is stored at mongolab, served by a HTTP/REST application hosted on heroku; a minimalistic Google Chrome extension is provided to facilitate bookmarks collection while browsing.

You can use the public (and free) store by submitting your email in the following form, or grab a copy of the source code (of both the server application and Chrome extension) from GitHub, possibly subscribe (for free) with mongolab and heroku and roll-out your own (private) store.
